1. Start MySQL and create a database named "eventplanning".

2. Import the given SQL file to the created database.

3. Open the terminal and navigate to the backend directory (cd backend)

	3.1. Execute the command "npm init -y"
	3.2. again Execute "npm install express mysql2 cors body-parser bcryptjs" (This will install the dependencies)
	3.3. start the server by executing the command "node server.js"

4. Navigate to the frontend directory (cd frontend)
	
	4.1. Execute the command "npm i". (This will install the dependencies)
	4.2. Execute "npm run dev" to run.
	4.3. Follow the link provided by the terminal to experience the Event Manager App.
